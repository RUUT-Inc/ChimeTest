# ChimeTest

## 概要
amazon-chime-sdk-jsを使ったプロダクト作成時の機能チェックを目的としたリポジトリです

## ブランチ
|ブランチ名|用途|
|---|---|
|develop|機能チェック用|
|feature/for_debug|カスタマーサポートに送る用. 現状問題の発生が観測できる最小構成|
## Build Setup

```bash
cp .env.example .env
yarn install
yarn build
yarn start
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
